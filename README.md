# System Monitor

Простая утилита для мониторинга состояния системы в реальном времени с выводом информации в консоль и сохранением в лог-файл.

## Описание

Скрипт выполняет две основные функции:
1. **Первичный сбор информации** о компонентах системы при запуске (RAM, диски, процессор, видеокарта)
2. **Непрерывный мониторинг** загрузки CPU, RAM и температуры GPU с записью в лог-файл

## Возможности

- **Оперативная память (RAM)**: информация о производителе, объеме, скорости, типе и слотах
- **Физические диски**: отображение имени, типа носителя, статуса работы, состояния здоровья и размера
- **Процессор (CPU)**: название, количество ядер/потоков, максимальная и текущая частота
- **Видеокарта (GPU)**: название, производитель, объем видеопамяти, разрешение, версия драйвера
- **Мониторинг в реальном времени**:
  - Загрузка CPU (%)
  - Использование RAM (%)
  - Температура GPU (°C)
- **Система предупреждений**: уведомления при превышении заданных пороговых значений
- **Автоматическое логирование**: все данные сохраняются в файл `start_times.txt`
